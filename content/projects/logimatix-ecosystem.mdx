## El problema

Las empresas pierden horas en tareas repetitivas: gestión de datos, respuestas a clientes,
coordinación entre sistemas que no se hablan entre sí. Cada proceso manual es un cuello de
botella y una fuente de errores.

Logimatix nace para resolverlo con **automatización inteligente operada sobre infraestructura propia**:
no un script suelto, sino una plataforma.

## La arquitectura

El sistema corre íntegramente sobre **Oracle Cloud**, con control total del entorno:

- **Compute**: Ubuntu VMs administradas directamente (hardening, actualizaciones, monitoreo)
- **Reverse proxy**: Nginx como capa de entrada, TLS y enrutamiento de servicios
- **Datos**: PostgreSQL como base transaccional de flujos y clientes
- **Orquestación**: n8n self-hosted para los flujos de automatización
- **Inteligencia**: APIs ML/LLM integradas a los flujos para clasificación y generación

### Integraciones en producción

Los flujos conectan sistemas reales de negocio:

- **CRM**: sincronización de contactos y estados vía API
- **Pagos**: Stripe y PayPal (cobros, confirmaciones, conciliación)
- **Notificaciones**: email y SMS disparados por eventos de negocio

## Decisiones técnicas

- **Self-hosted sobre Oracle Cloud en vez de SaaS**: control de datos, costo predecible
  y libertad para integrar APIs LLM sin límites de plataforma.
- **n8n como orquestador**: flujos versionables y reutilizables, con lógica custom en
  Python donde el nodo estándar no alcanza.
- **PostgreSQL como única fuente de verdad**: cada ejecución de flujo queda registrada
  para trazabilidad y depuración.

## Impacto real

- **50+ flujos de automatización** creados para clientes
- **Más de 70% de reducción** en procesos manuales de los clientes
