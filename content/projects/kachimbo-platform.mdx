## El contexto

Los estudiantes preuniversitarios necesitan **contenido dinámico, simulacros de examen y seguimiento de su progreso** en un solo lugar. Muchas soluciones son fragmentadas o dependen de infraestructura de terceros con costos altos y poco control.

Kchimbo nace como **plataforma EdTech** para ofrecer una experiencia educativa integrada, desplegada en infraestructura propia para tener control total sobre rendimiento, datos y escalabilidad.

## La solución

Diseñé y co-fundé Kchimbo como plataforma inteligente para preuniversitarios:

- **Contenido dinámico** adaptado al ritmo del estudiante
- **Simulacros de examen** optimizados para practicar bajo condiciones reales
- **Tracking de progreso** para que alumnos y docentes vean avances en tiempo real
- **Arquitectura escalable** desplegada en cloud para soportar picos de uso

## Infraestructura y stack

Todo el sistema está desplegado en **Oracle Cloud** con control total sobre el entorno:

- **Servidores**: VMs con Linux Ubuntu para backend y servicios
- **Web**: Next.js para la aplicación web (SSR, rutas dinámicas, optimización)
- **Proxy y estático**: Nginx para reverse proxy, SSL y entrega de assets
- **Contenedores**: Docker para consistencia entre entornos y despliegues
- **Bases de datos y APIs**: PostgreSQL y APIs propias para contenido y tracking

Esta arquitectura permite iterar rápido, hacer despliegues seguros y escalar cuando crece la base de usuarios.

## Reconocimiento

Kchimbo fue **finalista en Startup UTP**, compitiendo con más de 300 equipos y con un premio de **$10,000 USD** en juego. El proyecto valida la necesidad de herramientas EdTech bien construidas y orientadas a resultados medibles.

## Próximos pasos

El roadmap incluye ampliar tipos de simulacros, mejorar analytics para instituciones y seguir optimizando la experiencia de estudio en dispositivos móviles.
